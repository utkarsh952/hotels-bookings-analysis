# Hotel Bookings Analysis | Apr 2024 – Apr 2025

Exploratory data analysis of 13 months of hotel booking data across 
10 US cities, 3 booking channels, and 4 room types.

## Key Findings
- Travel Agent channel cancels at 28% vs Web at 17.6%
- Q3 has the highest cancellation rate (25.7%) despite lower revenue
- Travel Agent × Summer = 34.4% cancellation — the highest-risk segment
- Revenue flat at ~₹49M/month — growth capped without cancellation intervention

## Tools Used
Python · Pandas · Matplotlib · Seaborn
